# Sistema Ecopraça

Sistema de cadastro de moradores para associações.

## Funcionalidades
- Cadastro de moradores
- Campo NIS
- Identificação de participação em outra associação
- Relatórios simples

## Tecnologias
- Python
- Streamlit
- SQLite

## Como executar
```bash
py -3.11 -m streamlit run sistema_associacao_bairro_visual.py
